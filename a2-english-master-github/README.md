# A2 English Master 🚀

Mini-app per esercitarsi nel Writing A2 con:

- 📚 Studio Unit
- 🎓 Modalità esame con timer 40 minuti
- 🧠 Correzione base di grammatica e spelling
- 📖 Grammar Book
- 📊 Progressi salvati nel browser
- ✍️ Conteggio parole 120–150

## Come pubblicarlo su GitHub Pages

1. Crea una nuova repository su GitHub.
2. Carica tutti i file di questa cartella.
3. Vai su **Settings** → **Pages**.
4. In **Build and deployment**, scegli:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Salva.
6. Dopo qualche minuto GitHub ti darà il link pubblico.

## File principale

Il file principale deve chiamarsi:

```text
index.html
```

Non rinominarlo, altrimenti GitHub Pages potrebbe non aprire subito l'app.

## Nota importante

La correzione è basata su regole JavaScript offline. Riconosce molti errori tipici A2, ma non è un vero correttore AI tipo Grammarly.
