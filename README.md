## Prima di caricare

In Supabase SQL Editor esegui:

`SUPABASE_V17_SETUP.sql`

Poi controlla Vercel:

- `GEMINI_API_KEY`
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
