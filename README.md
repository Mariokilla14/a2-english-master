# 🚀 A2 English Master PRO

Versione professionale e modulare dell'app per preparare il Writing A2.

## Struttura

```text
index.html
css/style.css
js/app.js
js/grammarEngine.js
js/exam.js
js/progress.js
js/dataLoader.js
data/units.json
data/grammar.json
data/common_mistakes.json
data/spelling.json
data/tips.json
manifest.json
service-worker.js
```

## Come caricarlo su GitHub

1. Apri la repository `a2-english-master`.
2. Carica tutti i file e le cartelle contenuti in questo ZIP.
3. Fai commit.
4. GitHub Pages aggiornerà il sito automaticamente.

## Nota

Questa è una PWA-ready app: può funzionare anche offline dopo il primo caricamento.
Il correttore è ancora rule-based, ma ora è organizzato in moduli e può essere ampliato facilmente.
